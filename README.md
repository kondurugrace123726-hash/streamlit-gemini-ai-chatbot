# streamlit-gemini-ai-chatbot
AI chatbot built with Streamlit and Google Gemini API using secure environment variable management.

An AI-powered chatbot built using **Streamlit** and **Google Gemini API** with secure environment variable management.

## 🚀 Features

- ⚡ Fast responses using Gemini 2.5 Flash
- 🔐 Secure API key handling using '.env'
- 🎯 Simple and clean UI with Streamlit
- 🧠 Answers questions about tech, coding, and AI
- 🛠 Beginner-friendly and production-ready

---

## 📦 Tech Stack

- Python
- Streamlit
- Google Generative AI (Gemini API)
- dotenv
